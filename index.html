Img2GPS
Trained a Convolutional Neural Network to map images to GPS coordinates. Fine tuned EfficientNetV2 for the task. Before tuning, RMSE was around 0.0007 after longitude and latitude normalization. After extensive hyperparameter tuning using grid search, random search and k-fold cross validation, RMSE was reduced to 0.000219. With the testing data this is around 42 meters off on average.

RISC-V Processor
Created a custom RISC-V processor in Verilog. This includes a 32-bit RISC-V processor with a 5-stage pipeline. It includes a custom instruction set and a custom memory system utilizing a AXI4-Lite interface for the cache. The pipeling system uses a custom 8 stage divider streamlining the execution of further instructions. Processor passes the Dhrystone benchmark and was able to be synthesized into a FPGA.
